## Overview
This repository contains a Python implementation and mathematical study of the **PageRank Algorithm**. While most implementations focus solely on the ranking, this project performs a deep dive into the **convergence properties** of the algorithm. It specifically compares the theoretical error bounds (derived from the damping factor) against the actual decay of the error norm and the second-largest eigenvalue ($\lambda_2$).

## Features
- **Transition Matrix Calculation**: Automatically handles "dangling nodes" (pages with no outgoing links) by redistributing probability across the network.
- **Power Iteration**: An efficient implementation of the power method to find the stationary probability vector (the PageRank vector).
- **Convergence Analysis**: 
    - Computes the $L_1$ norm error at each iteration.
    - Calculates the theoretical convergence rate $c = 1 - m$.
    - Performs an **Eigenvalue Analysis** to verify that the actual convergence rate is governed by the second-largest eigenvalue ($|\lambda_2| < c$).
- **Data Support**: Includes a loader for the `hollins.dat` web graph dataset.

## Mathematical Context
The algorithm computes the PageRank vector $q$ such that $Mq = q$, where $M$ is the Google Matrix:
$M = (1-m)A_{norm} + \frac{m}{n}\mathbf{1}\mathbf{1}^T$

This project demonstrates that the power iteration converges at a rate determined by the subdominant eigenvalue. The script visualizes this by plotting the error $||q^{(k+1)} - q^{(k)}||_1$ on a logarithmic scale.
